# Sorteio-dgLindo
Este é um programa simples em Python que realiza o sorteio de números aleatórios e apresenta algumas estatísticas básicas sobre os números sorteados.

O usuário escolhe quantos números deseja sortear, e o programa retorna:

📊 A lista de números sorteados

🔝 O maior número

🔻 O menor número

📉 A média dos números

Após o sorteio, o usuário pode escolher refazer o sorteio ou encerrar o programa.

🧠 Funcionalidades

O programa possui três funções principais para análise dos números sorteados:

media()

Calcula e exibe a média dos números armazenados na lista.

maior()

Exibe o maior número presente na lista.

menor()

Exibe o menor número presente na lista.

⚙️ Tecnologias Utilizadas

Python 3

Bibliotecas padrão:

random

math

datetime

time

▶️ Como Executar

Certifique-se de ter o Python 3 instalado.

Baixe ou clone este repositório.

Execute o arquivo Python:
